# PinOne Post-Baller Activation Toolkit

Post-Baller hardware activation and validation toolkit for CSD PinOne virtual pinball cabinets.

## Scope (v1)
- CSD PinOne controller
- 10 solenoids
- SSF 7.1
- LED matrix / addressable LEDs

## What this repo does NOT include
- No tables
- No ROMs
- No copyrighted media
