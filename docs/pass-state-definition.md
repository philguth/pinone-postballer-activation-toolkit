# PASS State Definition (v0.1)

A cabinet is considered "Activated" when all systems below meet PASS criteria.

---

# Windows

- High Performance mode enabled
- USB selective suspend disabled
- Fast Startup disabled
- Sleep disabled
- Playfield set as primary display

---

# PinOne Controller

- Detected consistently
- All cabinet buttons register
- Plunger axis stable and responsive

---

# VPX

- Inputs mapped correctly
- Plunger calibrated
- Test table runs 2+ minutes without input drop

---

# DOF Outputs

- Output01–Output10 verified
- No stuck solenoids
- No idle chatter
- Shaker responds correctly

---

# LEDs

- Matrix test runs
- Addressable strips respond
- Brightness stable

---

# SSF 7.1

- Windows speaker test matches physical layout
- In-table sounds localized correctly

---

# Final Validation

- Health report generated
- All stages PASS or clearly documented FAIL with fix path

When all conditions above are met, the cabinet is considered fully Activated.
